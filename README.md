### Install

```bash
composer require wezom-agency/r2d2
```

---

### Setup

in your core app file:

```php
WezomAgency\R2D2::eject()->set('KEY', VALUE);
```

[See list of all settings ►](https://github.com/WezomAgency/r2d2/blob/master/docs/api/instance-set.md)
